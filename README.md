# 🎯 Codedex Course Tracker

An interactive, visual progress tracker for all **25 Codedex courses**, organized into 4 learning tracks. Built as a single self-contained HTML file — no install, no backend, just open and go.

🔗 **Live demo:** _(add your GitHub Pages link here after deploying)_

---

## ✨ Features

- ✅ Click any course to mark it complete / incomplete
- 📊 Overall progress ring + per-track progress bars
- 🎨 4 color-coded panels, one per track
- 💾 Progress saved automatically in your browser (`localStorage`) — survives refreshes
- 🔄 One-click reset if you want to start over
- 📱 Responsive layout — works on desktop, tablet, and mobile

## 🚀 Getting started

### Option 1 — Just open it
Download `index.html` and open it directly in any browser. That's it.

### Option 2 — Host it live (GitHub Pages)
1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Source: **Deploy from a branch** → `main` → `/ (root)`
4. Your tracker will be live at `https://<your-username>.github.io/<repo-name>/`

---

## 🛠️ Built with

- HTML5
- CSS3 (custom properties, grid layout)
- Vanilla JavaScript (no frameworks, no dependencies)
- `localStorage` for persistence

---

## 📝 Notes

- Progress is stored **per-browser**, not in the cloud — clearing browser data will reset it.
- Course list reflects the official [Codedex](https://www.codedex.io) catalog.
